# Typing Test Project

# Project Description
This is a simple Java project.  
The program allows users to:
- register an account
- log in
- reset password
- take a typing test
- view test results
- read the About Us section

# Features
- Welcome menu screen
- User registering screen
- User reset password screen
- User login screen
- List of test results for a user
- User typing test screen
- About Us screen

# Project Structure
Example project files:

- `Main.java`
- `WelcomeMenu.java`
- `RegisterUser.java`
- `ResetPassword.java`
- `LoginUser.java`
- `TestResults.java`
- `TypingTest.java`
- `AboutUs.java`
